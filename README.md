download:
git clone git@github.com:presk0/colinion.git

go directory:
cd colinion 

compile:
make

try it:
./colinion

using a text file as a map:
./colinion map.txt

Feel free to modify the ALGO.c file to create your own (src folder)
Cells type must be initialised in config.h (include folder)
